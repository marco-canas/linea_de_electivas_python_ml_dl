## 2. Relación con el Perfil

El curso proporciona fundamentos de programación, diseño de algoritmos y
escritura de programas en Python y SQL, aplicados a la solución de
problemas, incluyendo análisis de datos.

------------------------------------------------------------------------

## 3. Intencionalidades Formativas

Introducir los elementos fundamentales de la programación útiles para el
análisis de datos y la construcción de modelos, estudiando instrucciones
básicas, estructuras de control, definición de funciones y estructuras
de datos.

------------------------------------------------------------------------

## 4. Aportes a la Formación Integral e Investigación

El curso aporta elementos lógicos esenciales para cursos posteriores de
Estadística y Matemáticas. Prepara al estudiante para usar el computador
como herramienta fundamental en análisis de datos y modelación teórica y
aplicada.

------------------------------------------------------------------------

# 5. Descripción de Contenidos

## **Unidad 1: Elementos de programación**

-   Introducción al curso\
-   Lenguajes de programación\
-   Introducción a Python\
-   Tipos de datos, operadores, estructuras de datos estáticas y
    dinámicas

## **Unidad 2: Estructuras de control**

-   Estructuras condicionales simple y compuesta\
-   Condicionales múltiples y anidadas\
-   Iteración definida e indefinida

## **Unidad 3: Funciones**

-   Funciones incorporadas\
-   Módulos y submódulos\
-   Definición de funciones, ámbito, problemas de aplicación

## **Unidad 4: SQL y visualización**

-   Fundamentos de SQL\
-   Creación, filtros, particionamiento\
-   Funciones, subconsultas, unions, joins

# 6. Metodología

**Estrategias didácticas:**\
Clases magistrales, aprendizaje basado en proyectos, talleres de
laboratorio, solución de problemas.

**Recursos:**\
Talleres, notas de clase, videos, bibliografía, ejercicios guiados y
autónomos.

**Rol del docente:**\
Crear ambientes de aprendizaje, orientar resolución de problemas,
tutorías y acompañamiento.

**Rol del estudiante:**\
Participar activamente, asistir a tutorías, presentar soluciones,
mantener trabajo autónomo.

**Internacionalización:**\
Bibliografía en inglés, uso de recursos internacionales, clases espejo,
enfoque intercultural.
